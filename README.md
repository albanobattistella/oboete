# Installation
```
git clone https://github.com/mariinkys/oboete.git
cd oboete
sudo just install
```

# Build
```
git clone https://github.com/mariinkys/oboete.git
cd oboete
cargo build
```

## Dependencies
- [libcosmic](https://github.com/pop-os/libcosmic?tab=readme-ov-file#building)
